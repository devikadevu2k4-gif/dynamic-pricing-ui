# Dynamic Pricing Management System 🚀

An advanced, reactive user interface built with **Angular 18** to manage complex pricing structures for SaaS platforms. This project demonstrates modern front-end capabilities, including real-time state management and dynamic data manipulation.

## 📋 Problem Statement
The goal was to create a "Dynamic Pricing Management" UI that allows administrators to:
1. Manage a list of pricing tiers (Basic, Standard, etc.).
2. Handle **nested data** (features within plans).
3. Apply **multi-layer pricing logic** (Base Price + Fixed Setup Fees + 12% Percentage Tax).
4. Utilize **Angular Signals** for immediate UI updates without page refreshes.

## ✨ Key Features
- **Signals-Based Reactivity:** Uses `signal` and `computed` for high-performance data binding.
- **Dynamic Tier Management:** Add or remove pricing cards instantly with the built-in "New Tier" generator.
- **Nested Object Handling:** Efficiently manages arrays within arrays for plan features.
- **Live Calculations:** A fully functional pricing engine that handles math for fixed and percentage-based charges.
- **Responsive Design:** A modern, mobile-friendly card layout built with CSS Flexbox and Grid.

## 🛠️ Tech Stack
- **Framework:** Angular 18 (Standalone Components)
- **Language:** TypeScript
- **State Management:** Angular Signals
- **Styling:** Modern CSS3
