# Dynamic Pricing Management UI 🚀

This project is a dynamic, card-based interface built with **Angular 18**. It allows users to manage pricing tiers with real-time tax and fee calculations.

## ✨ Key Features
- **Angular Signals:** Implemented for high-performance, reactive state management.
- **Dynamic Calculations:** Automatically calculates Total Price (Base + Setup Fee + 12% Tax).
- **Nested Data:** Uses structured JSON arrays to manage features within each plan.
- **Interactive UI:** Users can add new pricing tiers or remove existing ones instantly.

## 🛠️ Tech Stack
- **Framework:** Angular (v18+)
- **Logic:** TypeScript & Signals
- **Styling:** CSS3 with Flexbox/Grid
- **Version Control:** Git & GitHub
